Anneal.jl's Assets

Logo

Anneal.jl's logo aims to illustrate the annealing process with a progressive addition of the energy landscape.

Colors

The colors were chosen according to Julia's Reference for logo graphics¹. Text color matches the innermost shape and renders fairly well in both light and dark background themes.

Typography

MADETYPE Sunflower² Font was used. It was converted to a SVG path through the Google Font to Svg Path³ online tool.
